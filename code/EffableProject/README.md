# Telegram Emotion Bot (v1)

Простейший Telegram-бот для отслеживания эмоционального состояния пользователя.

В первой версии бот:
- **раз в сутки** (в 21:00 по локальному времени сервера) отправляет пользователю сообщение:  
  "Привет! Как прошел твой день?"
- пишет **всем пользователям, которые хотя бы раз написали боту**
- принимает любые ответы пользователя (пока **без сохранения в базу данных**)

## Установка

1. **Клонируй или скопируй** проект в отдельную папку.
2. Убедись, что установлен **Python 3.10+**.
3. Установи зависимости:

```bash
pip install -r requirements.txt
```

## Настройка токена

1. Создай Telegram-бота через `@BotFather` и получи **BOT TOKEN**.
2. Открой файл `bot/config.py`.
3. Найди строку:

```python
BOT_TOKEN = "PUT_YOUR_TELEGRAM_BOT_TOKEN_HERE"
```

4. Замени строку на свой реальный токен, например:

```python
BOT_TOKEN = "1234567890:ABCDEF_your_real_token_here"
```

Токен используется из **одного места** (`config.py`) и больше нигде не дублируется.

## Как запустить бота

Из корневой папки проекта (где лежит `requirements.txt`) выполни:

```bash
python -m bot.main
```

После запуска:
- бот начнет принимать сообщения от пользователей
- всех пользователей, которые написали боту хотя бы один раз, он будет считать «подписчиками»
- каждый день в **21:00** по локальному времени сервера бот отправит всем таким пользователям вопрос:

> Привет! Как прошел твой день?

Бот является **MVP-проектом** и специально сделан максимально простым, чтобы его было легко дорабатывать.
